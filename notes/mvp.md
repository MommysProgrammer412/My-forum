# MVP - Базовый функционал

## По ролям
## 1. Польлзователь

**Регистрация**
- При регистрации указать: имя пользователя, пол, дата рождения (без года), почта, пароль, пароль еще раз.
- При регистрации нажать: зарегистрироваться.

**Вход**
- При входе указать: почту, пароль.
- При входе нажать: войти, забыл пароль.

**Восстановление пароля**
- При восстановлении пароля указать: почту.
- При восстановлении пароля нажать: получит письмо.
- Уведомление: `на вашу почту было отправлено письмо с сылкой для восстановления пароля`.

**Разделы**
- Профиль, главная, поиск, уведомления, реакции, настройки.
**Профиль**
- Отображается: аватарка, имя пользователя, статус, количество постов/подписчиков/подписок, список постов/ретвитов пользователя.
- Возможно: создать пост, взаимодействие с постами, прокрутка вниз.
**Главная (открывается по умолчанию при входе)**
- Отображается: посты подписок.
- Возможно: взаимодействие с постами, прокрутка вниз.
**Поиск**
- Отображается: поисковая строка.
- Возможно: поиск профиля и постов с хэштэгом.
**Уведомления**
- Отображается: ретвит/лайки/комментарии поста пользователя, день рождения подписок.
**Реакции**
- Отображается: лайкнутые посты и комментарии.
- Возможно: взаимодействие с постами/комментариями, прокрутка вниз.
**Настройки**
- Отображается: информация о профиле пользователя.
- Возможно: выйти/удалить аккаунт, изменить почту/пароль, изменить/удалить аватарку, изменить имя пользователя, изменить статус.
**Посты**
- При создании возможно: добавлять хэштэги, прикреплять фото/фидео/докуменеты/аудиофайлы, добавлятьтекст поста.
- После создания можно: писать комментарии под ним, удалить/редактировать свой пост, поставить/снять лайк, ретвитнуть в свой профиль (с возможностью дополнить ретвит информацией как обычный пост), закрепить пост/ретвит в своем профиле.
**Комментарии**
- Возможно: писать/редактировать/удалять свои комментарии, поставить/снять лайк.

## 2. Администратор

**Имеет весь функционал пользователя и возможность**
- Заблокировать/разблокировать: пользователя, пост, комментарий, хэштэг и посты с этим хэштэгом.
